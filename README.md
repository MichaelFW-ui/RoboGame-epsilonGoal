# Team epsilonGoal 电控设计

## 前言

本机器人控制程序按照2021年中国科学技术大学Robo Game机器人给定任务设计。本程序的开发和使用旨在学习机器人控制系统、电机控制理论和常用传感器的使用，仅供学习和娱乐用途使用。

本程序中可能使用部分公开API或他人自行设计的接口代码，一般情况下是有所标注。如有侵权，联系本人即可。
